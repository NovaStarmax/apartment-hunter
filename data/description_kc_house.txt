=== KC HOUSE DATA - RÉSUMÉ DU DATASET ===

Dataset: King County House Sales dataset
Période: Mai 2014 - Mai 2015
Nombre total d'observations: 21,613 propriétés
Source: King County, Washington State, USA

=== COLONNES ET TYPES ===

1. id (int64)
   - Identifiant unique pour chaque propriété vendue
   - Données manquantes: 0

2. date (object/datetime)
   - Date de vente de la propriété
   - Format: YYYYMMDDTHHMMSS
   - Données manquantes: 0

3. price (float64)
   - Prix de vente de la propriété en USD
   - Données manquantes: 0

4. bedrooms (int64)
   - Nombre de chambres
   - Données manquantes: 0

5. bathrooms (float64)
   - Nombre de salles de bain (0.5 = toilettes sans douche)
   - Données manquantes: 0

6. sqft_living (int64)
   - Surface habitable en pieds carrés
   - Données manquantes: 0

7. sqft_lot (int64)
   - Surface du terrain en pieds carrés
   - Données manquantes: 0

8. floors (float64)
   - Nombre d'étages (ex: 1.0, 1.5, 2.0)
   - Données manquantes: 0

9. waterfront (int64)
   - Vue sur l'eau (0 = non, 1 = oui)
   - Données manquantes: 0

10. view (int64)
    - Qualité de la vue (0 à 4)
    - Données manquantes: 0

11. condition (int64)
    - État général de la propriété (1 à 5)
    - Données manquantes: 0

12. grade (int64)
    - Niveau de construction et de design (1 à 13)
    - Données manquantes: 0

13. sqft_above (int64)
    - Surface habitable au-dessus du sol en pieds carrés
    - Données manquantes: 0

14. sqft_basement (int64)
    - Surface du sous-sol en pieds carrés
    - Données manquantes: 0

15. yr_built (int64)
    - Année de construction
    - Données manquantes: 0

16. yr_renovated (int64)
    - Année de rénovation (0 si jamais rénové)
    - Données manquantes: 0

17. zipcode (int64)
    - Code postal
    - Données manquantes: 0

18. lat (float64)
    - Latitude
    - Données manquantes: 0

19. long (float64)
    - Longitude
    - Données manquantes: 0

20. sqft_living15 (int64)
    - Surface habitable moyenne des 15 voisins les plus proches
    - Données manquantes: 0

21. sqft_lot15 (int64)
    - Surface de terrain moyenne des 15 voisins les plus proches
    - Données manquantes: 0

=== NOTES ===
- Dataset généralement très propre avec peu ou pas de valeurs manquantes
- Pour vérifier les données manquantes réelles, exécuter: df.isnull().sum()
